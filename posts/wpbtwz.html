WordPress不同分类调用不同文章模板

问题：

老师你好，我是学做网站论坛的一个学员，我学习了wordpress建站教程后，使用Wordpress程序建了一个网站，我想不同的分类目录调用不同的文章模板。例如：产品列表分类下的文章与新闻列表分类下的文章调用各个不同的文章模板。这应该怎么实现呢？

回复：

不同分类调用不同的文章模板，可以给分类目录添加属性，可以通过代码的方式去实现，并且可以在文章页面去调用自己添加的分目的属性。

		
<?php

		$post = $wp_query->post;

		if ( in_category(array("1","2")) || post_is_in_descendant_category("1")) {

		include(TEMPLATEPATH . '/single-pro.php');

		}

		else {

		include(TEMPLATEPATH . '/single-wz.php');

		} ?>

这样的代码去判断分类的ID号来调用不同的模板。
